## Prompt Guidelines

### Style Keywords
- `modern`, `clean`, `professional`, `developer tool`
- `dark UI`, `dark theme`, `dark background`
- `orange accent`, `#F97316`
- `flat design`, `minimal`, `iconic`

### Avoid
- Generic SaaS styles
- Big bubbly buttons
- Childish mascot overload
- Light/white backgrounds for dark theme assets
- Text in images (use overlays for text)

### Color References
- Primary: `#F97316` (orange)
- Secondary: `#F59E0B` (amber)
- Background: `#09090B` (near-black)
- Panel: `#111113` (dark gray)
- Text: `#FAFAFA` (white)
